Consulta Tus Cotizaciones

Bienvenido a tu sección de cotizaciones. Aquí puedes consultar tus cotizaciones vigentes. Recuerda revisar el periodo de vigencia de cada cotización para tramitar tus pedidos a tiempo.

Para ver los detalles de una cotización, selecciona la que te interese.

Las cotizaciones que incluyan cambios de dirección de envío o productos controlados serán procesadas por un Asesor Comercial y no aparecerán aquí. Un asesor se pondrá en contacto contigo para dar seguimiento.

Cuando estés listo para proceder, selecciona "Tramitar Pedido" para agregar partidas a una orden de compra.

Si tienes dudas, contacta a nuestro equipo de ventas en ventas@proquifa.com
